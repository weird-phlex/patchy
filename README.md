# Weird Phlex

A package manager for "soft dependencies", mainly aimed at UI components

## Installation

Adding to a gem:

```ruby
# my-cool-gem.gemspec
Gem::Specification.new do |spec|
  # ...
  spec.add_dependency "weird_phlex"
  # ...
end
```

Or adding to your project:

```ruby
# Gemfile
gem "weird_phlex"
```

### Supported Ruby versions

- Ruby (MRI) >= 3.1.0
